# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial release
- `ClickCheckClient` with full API support
- URL scanning functionality with evaluation modes (default, strikt, grosszuegig)
- Blacklist check and reporting
- Credit balance management
- Reward system for accepted blacklist reports (0.001€ per report)
- Comprehensive error handling with custom exceptions
- Rate limit handling with retry information
- Full TypeScript-style docstrings

### Features
- Plan-based rate limiting support (FREE: 10/min, PRO: 60/min, BUSINESS: 300/min)
- Affordable pricing (0.012€ per scan)
- Easy-to-use Pythonic API
